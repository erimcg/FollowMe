1. Logging In:

When logging into the application, you must enter your username and password. When these have been entered correctly, click the “log in” button. If the user is already logged in they will not be prompted to log in again. The application will take the user to the main screen if you are already logged in


2. Registering For An Account:

You will enter the user name you wish to use, your e-mail address, your first and last name, then you will input the password you wish to use into the password field and confirm your password. You can then create your account by pressing the “Register” button. Please keep in mind only one person can create an account per email and all usernames are unique.


3. Main Screen:

The main screen will show your current location through GPS and display your location on the map. There will be a tap with three lines on the top left corner of the screen and it will pull out slider menu with the following options: Load Excursion, Start Recording Route, Add Observation, Stop Recording Route, Edit Excursion, Save Excursion, Log Out, and Exit. By default an excursion is created for the user and by using these menu options a user may add route points and observations. For more information on what each menu option does, please look at the respective section on this page.


4. Loading Excursion:

By clicking on load excursion in the menu bar the application gives you a list of excursions and then by clicking on one of these it load the excursion. If the excursion you are on has changes the application will ask if you want to save those changes. The application will update the list of available excursions if you are connected to the internet. You can scroll through the list of excursions and choose another excursion or click “cancel” to go back to the main screen. You can then view the excursion route and the observation points on that route.


5. Start Recording Routes:

When you press red circle at the top right of the screen on the menu bar at the top of the screen, using gps the app will start recording where the user is traveling and plotting it on your map with a red line using GPS.


6. Stop Recording Routes:

If you want to stop recording your route on your map with GPS then click on the square button at the top right of the screen.


7. Adding Observation:

When you click on “add observation” on the menu, the screen will display a form. This form has a section to write the title of the observation, a section to add a description of your observation, and a button labeled “Cancel” and “Save”. The date and time will be saved when you press the "save" button. If you do not wish to save your observation press the "cancel" button.


8. Viewing Observations:

If you click on an observation a menu will be displayed showing you the title, description and GPS location of the observation. If you are the one who created the excursion you will see an “edit” button which allows you to add changes to the observation see the “Edit Observation” help section for further help. Pressing “back” will bring the user back to the map.


9. Editing Observations:

The title, obeservation description and GPS location will already be filled. You can edit these fields as needed. You can then press the button labeled “Save” to save the observation. The button labeled “Cancel” will exit the edit screen without applying changes and the “Delete” button will remove the observation completely.


10. Editing Excursion:

If you press “edit excursion” the pull down menu will be displayed with the excursion name, description. You will also see two toggle buttons which give you the option of walking or driving depending on the route and allow you to toggle between public and private depending on whether or not you want others to see the route. You can then press “cancel” to exit without changing anything or “save” to apply the changes you have made.


11. Saving Excursions:

When saving an excursion, you will need to give the excursion a unique name different from all other excursion names in order to save the excursion successfully. If connected to the internet and an excursion is public your excursion will be saved in the cloud. If you are not connected to the internet the excursion will be saved but not uploaded to our server.


12. Logging Out:

To log out press the menu button at the top of the screen and click "log out" and you will be taken to the log in screen.
